# Machine Learning Prognostics for Genetic Disorder Prediction

This project focuses on predicting genetic disorders using machine learning techniques. It applies data preprocessing, exploratory data analysis, and multiple classification algorithms to analyze genetic and clinical data and generate reliable predictions.

---
## Project Overview

The goal of this project is to build and evaluate machine learning models that can predict the likelihood of genetic disorders based on input data. The project emphasizes early detection, improved accuracy, and support for personalized healthcare decision-making.

---
## Modules Included

### Data Preprocessing
- Handling missing values and duplicates
- Data type validation and cleaning

### Exploratory Data Analysis & Visualization
- Understanding data distributions
- Identifying patterns and relationships using plots

### Machine Learning Models
- Bagging Classifier
- Random Forest Classifier
- Naive Bayes Classifier

### Model Evaluation
- Performance comparison of classifiers
- Accuracy and robustness analysis
